## 为什么学习 JavaScript?

JavaScript 是 web 开发人员必须学习的 3 门语言中的一门：

  1. **HTML** 定义了网页的内容
  2. **CSS** 描述了网页的布局
  3. **JavaScript** 控制了网页的行为



本教程是关于 JavaScript 及介绍 JavaScript 如何与 HTML 和 CSS 一起工作。
